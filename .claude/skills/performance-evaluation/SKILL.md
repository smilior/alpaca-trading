# パフォーマンス評価スキル

## このスキルの目的

トレーディングパフォーマンスを正しく測定し、意思決定に使える形で評価せよ。「なんとなく儲かっている」ではなく、統計的に根拠のある評価を行う。

## なぜ正しい評価が必要か

### 「勝率が高い＝良い戦略」ではない理由

```
戦略A: 勝率90%、平均利益$10、平均損失$100
→ 期待値 = 0.9 × $10 - 0.1 × $100 = $9 - $10 = -$1
→ 勝率90%でも期待値マイナス。やればやるほど損する。

戦略B: 勝率30%、平均利益$100、平均損失$20
→ 期待値 = 0.3 × $100 - 0.7 × $20 = $30 - $14 = +$16
→ 勝率30%でも期待値プラス。長期的に利益が出る。
```

**教訓**: 勝率だけでなく、期待値（Expectancy）で戦略を評価せよ。

## 最低限追跡すべきKPI一覧

| KPI | 意味 | 基準 |
|-----|------|------|
| シャープレシオ | リスク調整後リターン | > 1.0 で良好 |
| ソルティノレシオ | 下振れリスク調整後リターン | > 1.5 で良好 |
| 最大ドローダウン | ピークからの最大下落率 | < 15% を目標 |
| プロフィットファクター | 総利益 ÷ 総損失 | > 1.5 で良好 |
| 勝率 | 勝ちトレードの比率 | 戦略による |
| ペイオフレシオ | 平均利益 ÷ 平均損失 | > 1.5 で良好 |
| 期待値 | 1トレードあたりの期待利益 | > 0 が必須 |
| CAGR | 年率複利成長率 | > ベンチマーク |

各KPIの計算式は [references/kpi-definitions.md](references/kpi-definitions.md) を参照。

## ベンチマーク比較が必須である理由

```
❌ 悪い評価:
「先月+5%のリターンだった。良い成績だ！」
→ 同じ期間にS&P500が+8%なら、実質的には負けている

✅ 良い評価:
「先月+5%。同期間のS&P500は+3%。
超過リターン（α）は+2%。
ただしボラティリティが1.5倍なので、
リスク調整後では若干劣後。」
→ ベンチマークとの相対評価ができている
```

### 推奨ベンチマーク

| ベンチマーク | ティッカー | 用途 |
|-------------|-----------|------|
| S&P 500 | SPY | 米国大型株の標準ベンチマーク |
| NASDAQ 100 | QQQ | テック中心のポートフォリオ |
| Russell 2000 | IWM | 小型株ポートフォリオ |
| リスクフリーレート | - | シャープレシオの計算 |

## 評価のタイミング

| 頻度 | 目的 | 主要KPI |
|------|------|---------|
| 日次 | 異常検出、リスク確認 | 日次P&L、ドローダウン、ポジション数 |
| 週次 | 短期トレンドの把握 | 週次リターン、勝率、取引回数 |
| 月次 | 戦略の有効性検証 | シャープレシオ、ベンチマーク比較、全KPI |

レポートテンプレートは [references/report-templates.md](references/report-templates.md) を参照。

## 評価の注意点

### 短期間の成績で判断しない

```
❌ 悪い判断:
「2週間で+8%！この戦略は最高だ！」
→ 2週間では統計的に何も言えない。たまたまかもしれない。

✅ 良い判断:
「3ヶ月間、100回以上の取引で、
シャープレシオ1.2、プロフィットファクター1.6。
異なる市場環境（上昇相場と横ばい相場）でもプラス。
ただし下落相場のデータがまだないため、注意が必要。」
→ 統計的に意味のある期間と取引回数で評価
```

### 生き残りバイアスに注意

```
❌ 「3つの戦略を並行テストして、最も良かった1つを採用」
→ 3つ中1つが偶然良い結果を出す確率は高い

✅ 「事前に定めた基準を、3つの戦略全てに同じ基準で適用し、
基準を満たした戦略のみを採用」
→ 基準を事前に決めることで、結果に基づく選り好みを防ぐ
```

### ベンチマークの選び方

```
❌ 自分に都合の良いベンチマークを選ぶ
→ テック株ポートフォリオの成績をラッセル2000と比較
  （テックが上がった時期なら楽に勝てる）

✅ 投資ユニバースに合ったベンチマークを選ぶ
→ テック株ポートフォリオならNASDAQ 100（QQQ）
  または、単純にSPYを使う（最も一般的）
```

## パフォーマンス評価チェックリスト

評価レポートを作成する前に確認せよ：

- [ ] 十分な取引回数がある か（最低30回、推奨100回以上）
- [ ] 十分な期間がある か（最低1ヶ月、推奨3ヶ月以上）
- [ ] ベンチマーク（SPY）との比較をしている か
- [ ] リスク調整後リターン（シャープレシオ）を計算している か
- [ ] 最大ドローダウンを確認している か
- [ ] 異なる市場環境での成績を分析している か
- [ ] 手数料・スリッページを考慮している か
- [ ] 生き残りバイアスに注意している か
