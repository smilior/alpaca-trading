# 米国株市場リサーチスキル

## このスキルの目的

米国株市場の調査・分析を体系的に行え。トップダウンアプローチ（マクロ→セクター→個別銘柄）で市場を理解し、アルゴリズムトレーディングに適した銘柄を選定せよ。

## なぜ体系的なリサーチが必要か

- **ランダムに銘柄を選ぶと失敗する**: アルゴ取引に適さない銘柄（低流動性、ワイドスプレッド）を選ぶとスリッページで利益が消える
- **マクロ環境を無視すると危険**: 個別銘柄がどれほど良くても、マクロ環境（金利、景気サイクル）に逆らうと勝てない
- **セクター理解なしに分散はできない**: 「10銘柄に分散した」つもりが全てテック株なら、分散になっていない

## 市場分析の体系的アプローチ

### レベル1: マクロ分析

現在の市場環境を把握する。以下の指標をチェックせよ：

| 指標 | 確認内容 | データソース |
|------|----------|------------|
| FRB金利政策 | 利上げ/利下げサイクル | FRED |
| VIX（恐怖指数） | 市場のボラティリティ水準 | Alpaca / Yahoo Finance |
| イールドカーブ | 逆イールドならリセッション警戒 | FRED |
| S&P500のトレンド | 200日移動平均の上/下 | Alpaca Market Data |
| セクターパフォーマンス | どのセクターが強い/弱い | 各種ETF |

### レベル2: セクター分析

GICS 11セクターのうち、現在の環境で有利なセクターを特定する。
詳細は [references/sector-analysis.md](references/sector-analysis.md) を参照。

### レベル3: 個別銘柄スクリーニング

選択したセクター内で、アルゴ取引に適した銘柄を絞り込む。
詳細は [references/screening-criteria.md](references/screening-criteria.md) を参照。

## Alpacaで取得できるデータの範囲と限界

### 取得可能

| データ | エンドポイント | プラン |
|--------|---------------|--------|
| 株価（OHLCV） | `/v2/stocks/{symbol}/bars` | 無料（IEX） |
| リアルタイムquote | `/v2/stocks/{symbol}/quotes/latest` | 無料（15分遅延）/ 有料（リアルタイム） |
| Trade履歴 | `/v2/stocks/{symbol}/trades` | 無料 |
| スナップショット | `/v2/stocks/{symbol}/snapshot` | 無料 |
| ニュース | `/v1beta1/news` | 無料 |
| 企業情報 | 限定的 | - |

### 取得不可能（外部データソースが必要）

| データ | 推奨ソース |
|--------|-----------|
| 財務諸表（決算データ） | SEC EDGAR, Yahoo Finance API |
| アナリスト予想 | Yahoo Finance |
| マクロ経済指標 | FRED API |
| インサイダー取引 | SEC EDGAR |
| 機関投資家保有状況 | SEC 13F filing |
| オプションデータ | 別途データプロバイダー |

## 無料データソースの一覧と使い分け

| ソース | 用途 | レート制限 | 備考 |
|--------|------|-----------|------|
| Alpaca Market Data | 株価・出来高 | 200リクエスト/分 | メインのデータソース |
| Yahoo Finance (yfinance) | 財務データ・ヒストリカル | 明示的制限なし（非公式） | バックアップとして利用 |
| FRED API | マクロ経済指標 | 120リクエスト/分 | 金利・GDP・CPI等 |
| SEC EDGAR | 決算書・13F | 10リクエスト/秒 | 公式データ |
| Alpaca News API | ニュース | Alpacaのレート制限内 | センチメント分析用 |

詳細は [references/data-sources.md](references/data-sources.md) を参照。

```markdown
- VIX: [水準とトレンド]

- 市場レジーム: [ブル/ベア/レンジ]

### 2. セクター分析

- 弱いセクター: [下位3セクターとその理由]

### 3. 銘柄スクリーニング結果

|------------|----------|----------|------------|-----------|----------|
|            |          |          |            |           |          |

### 4. 推奨ユニバース
[最終的な投資対象銘柄リスト]

### 5. 注意事項・リスク
[マクロリスク、イベントリスク等]
```

## アンチパターン

```
❌ 悪いリサーチ:
- 「AAPL, TSLA, NVDA は有名だから取引しよう」
→ 人気だけで選んでいる。流動性・スプレッド・セクター集中を確認していない

❌ 悪いリサーチ:
- 「AIが流行っているからテック株全部買おう」
→ マクロ環境を無視。金利上昇局面でテック株は不利になりうる

✅ 良いリサーチ:
- マクロ: FRB利下げサイクル入り → 成長株に追い風
- セクター: テクノロジーとヘルスケアが相対的に強い
- 銘柄: 日次出来高100万株以上、スプレッド0.05%以内、
  時価総額$10B以上でスクリーニング → 15銘柄に絞り込み
- リスク: 選挙年のため、9-10月にボラティリティ上昇の可能性
```
